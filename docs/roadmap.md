# 路线图与演进

## Phase 0 - 概念验证（MVP）
- 仅支持 B 站。
- 完成博主管理、新视频发现、下载、下架检测、存储清理。
- 单机部署，SQLite + 本地文件存储。

## Phase 1 - 稳定与可观测性
- 任务重试、错误恢复、失败告警。
- 更完整的统计与报表。
- 可配置化调度策略与清理权重。

## Phase 2 - 多平台扩展
- 引入平台适配器接口。
- 接入抖音/快手/小红书作为独立适配器。
- 统一视频与博主模型。

## Phase 3 - 端侧应用
- 提供移动端/桌面端浏览与管理。
- 支持多用户与权限。
- 云端存储或分布式存储支持。
